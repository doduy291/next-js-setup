# Next.js setup

Starter code for a clean Next.js + TypeScript + ESLint + Husky project.

## Get started

- <h3>Install dependencies</h3>

```bash
# NPM
npm install
# Yarn
yarn
```

- <h3>Enable husky</h3>

```bash
# NPM
npm husky install
# Yarn
yarn husky install

// Ensure that you have got a Git repository, otherwise run this command:
git init
```

- <h3>Start dev server</h3>

```bash
# NPM
npm run dev
# Yarn
yarn dev
```

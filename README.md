# Next.js setup

Starter code for a clean Next.js + TypeScript + ESLint + Husky project.

## Get started

### Install dependencies

```bash
# NPM
npm install
# Yarn
yarn
```

### Enable husky

```bash
# NPM
npm husky install
# Yarn
yarn husky install
```

**_ NOTE: _** Ensure that you have got a Git repository, otherwise run this command: `git init`

### Start dev server

```bash
# NPM
npm run dev
# Yarn
yarn dev
```
